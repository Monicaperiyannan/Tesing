# QA Technical Assignment - Tichi Web Application

This repository contains the complete QA technical assignment for the **Tichi Web Application**.

##  Repository Structure

- **Task 1: Test Case Creation**
  - `TASK 1- SIGNUP AND LOGIN.xlsx`: Comprehensive test cases for Signup and Login modules.
- **Task 2: Defect Reporting**
  - `TASK2-DEFECT REPORT.docx`: Detailed bug report for the "Invalid Email Format Login" issue.
- **Task 3: Automation Testing**
  - `TICHI AUTOMATION TESTING.mp4`: A screen recording showing the automation script in action.

## 🛠 Tools Used
- **Manual Testing:** Microsoft Excel
- **Defect Reporting:** Microsoft Word
- **Automation:** Python, Selenium WebDriver
- **Version Control:** GitHub
